Kinematic Templates Cursor Manipulation Library

This library provides the mathematical functions for manipulating
cursor movement for my UIST 2008 publication.

Fung, R., Lank, E., Latulipe, C., and Terry, M. 2008.
Kinematic Templates: End-User Tools for Content-Relative Cursor Manipulations.
Proceedings of the 21st Annual ACM Symposium on User interface Software and Technology (UIST 2008) (Monterey, CA), 47-56. 

Please refer to my website for Java source code examples:
http://www.rhfung.com/core/Projects/KinematicTemplates 

Requires:

Microsoft Visual C# 2008 or higher

Target:

Windows Forms application

Includes:

/bin contains precompiled the library
/src includes the source code
/examples/CDPaint2 provides a simple drawing application
